About this project

This is a civic governance experiment. The PPPC is not a registered political party. Its manifesto, constitution, and bylaws are stored as version-controlled text files in a public GitHub repository. Changes are proposed as pull requests, reviewed openly, and validated by automated checks. The source of truth is always the repository.
