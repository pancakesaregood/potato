# Tokenomics

## Summary

This addendum defines a non-speculative model for civic tokens or credits if the PPPC chooses to pilot them.

## Design Principles

- Any token system must support civic participation rather than speculation.
- Tokens should not create unequal political rights.
- Rules for issuance and expiry must be plain and auditable.

## Technical Mechanisms

- Limit tokens to civic credits, pilot incentives, or transparent benefit routing.
- Publish issuance, redemption, and anti-hoarding rules before deployment.
- Ban any token design that implies the sale of voting power or legal status.

## Validation Targets

- Token balances and issuance events are auditable.
- Pilot participants understand the purpose and limits of tokens.
- No token mechanism changes formal democratic equality.
