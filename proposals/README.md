# Proposals

This directory stores future policy proposals, manifesto amendments, and
experimental governance drafts.

## Rules

- Use `templates/proposal_template.md`.
- Keep one proposal per file.
- Name files with a short, stable slug.
- Link proposals to any affected manifesto article or governance document.
- Include a real constitutional review note before seeking formal adoption.

## Example Filename

`2026-04-digital-referendum-pilot.md`
